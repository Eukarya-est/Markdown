# 15. 3Sum

\#Medium #Array #Sorting #Two pointers #Divide and Conquer

[toc]

## 1. Question

Given an integer array nums, return all the triplets `[nums[i], nums[j], nums[k]]` such that `i != j`, `i != k`, and `j != k`, and `nums[i] + nums[j] + nums[k] == 0`.

Notice that the solution set must not contain duplicate triplets.

 

**Example 1:**

> Input: nums = [-1,0,1,2,-1,-4]
> Output: [[-1,-1,2],[-1,0,1]]
> Explanation: 
> nums[0] + nums[1] + nums[2] = (-1) + 0 + 1 = 0.
> nums[1] + nums[2] + nums[4] = 0 + 1 + (-1) = 0.
> nums[0] + nums[3] + nums[4] = (-1) + 2 + (-1) = 0.
> The distinct triplets are [-1,0,1] and [-1,-1,2].
> Notice that the order of the output and the order of the triplets does not matter.

**Example 2:**

>Input: nums = [0,1,1]
>Output: []
>Explanation: The only possible triplet does not sum up to 0.

 **Example 3:**

>Input: nums = [0,0,0]
>Output: [[0,0,0]]
>Explanation: The only possible triplet sums up to 0.



**Constraints:**

- $$ 3 \leqq nums.length \leqq 3000 $$
- $$ -10^5 \leqq  nums[i] \leqq 10^5 $$

## 2. Solution

### 1) Two pointer

- Time Complexity: $$ O(n^2) $$
- Space Complexity: $$ O(|ans|) $$

#### A. C++

- Runtime: 35 ms
- Memory: 28.96MB
```C++
class Solution {
public:
    vector<vector<int>> threeSum(vector<int>& nums) {
        const int n = nums.size();
        if (n < 3) return {};

        vector<vector<int>> ans;

        ranges::sort(nums);

        for(int i = 0; i < n - 2; i++) {
            if (nums[i] > 0) break; //If the current smallest number is > 0, no triplet can sum to 0.
            if (i > 0 && nums[i] == nums[i - 1]) continue;

            int l = i + 1;
            int r = n - 1;
            while (l < r) {
                const int sum = nums[i] + nums[l] + nums[r];
                if (sum == 0) {
                    ans.push_back({nums[i], nums[l++], nums[r--]});
                    while(l < r && nums[l] == nums[l - 1]) l++;
                    while(l < r && nums[r] == nums[r + 1]) r--;
                } else if (sum < 0) {
                    l++;
                } else {
                    r--;
                };
            };
        };
        return ans;
    }
};
```

```Python
class Solution:
    def threeSumClosest(self, nums: List[int], target: int) -> int:
        nums.sort()
        closest = float('inf')
        n = len(nums)

        for i in range(n - 2):
            # Skip duplicates
            if i > 0 and nums[i] == nums[i - 1]:
                continue

            l, r = i + 1, n - 1

            # Pruning: check extreme sums
            min_sum = nums[i] + nums[l] + nums[l + 1]
            if min_sum > target:
                if abs(min_sum - target) < abs(closest - target):
                    closest = min_sum
                break

            max_sum = nums[i] + nums[r] + nums[r - 1]
            if max_sum < target:
                if abs(max_sum - target) < abs(closest - target):
                    closest = max_sum
                continue
			
            # Begin
            while l < r:
                s = nums[i] + nums[l] + nums[r]
                if abs(s - target) < abs(closest - target):
                    closest = s
                if s < target:
                    l += 1
                elif s > target:
                    r -= 1
                else:
                    return target
        return closest
```
### 2) Partitioning (Categorized search)

- Time Complexity: $$ O(n^2) $$
- Space Complexity: $$ O(n) $$

#### A. Python
- Runtime: 435 ms
- Memory: 24.51MB
```python
class Solution:
    def threeSum(self, nums: list[int]) -> list[list[int]]:
        ans = set()
        p, n, z = [], [], []
        for x in nums:
            if x > 0: p.append(x)
            elif x < 0: n.append(x)
            else: z.append(x)
        
        P, N = set(p), set(n)

        # Case1 (p, 0, n)
        if z:
            for x in P:
                if -x in N:
                    ans.add((-x, 0, x))
            if len(z) >= 3:
                ans.add((0, 0, 0))
        
        # Case2 (p1, p2, n)
        for x in range(len(p) - 1):
            for y in range(x + 1, len(p)):
                target = -(p[x] + p[y])
                if target in N:
                    ans.add(tuple(sorted((p[x], p[y], target))))

        # Case3 (n1, n2, p)
        for x in range(len(n) - 1):
            for y in range(x + 1, len(n)):
                target = -(n[x] + n[y])
                if target in P:
                    ans.add(tuple(sorted((n[x], n[y], target))))

        return [list(x) for x in ans]
```
