# 7. Reverse Integer

#Medium #Math

## 1) Question

Given a signed 32-bit integer `x`, return `x` *with its digits reversed*. If reversing `x` causes the value to go outside the signed 32-bit integer range $[-2^{31}, 2^{31} - 1]$, then return `0`.

**Assume the environment does not allow you to store 64-bit integers (signed or unsigned).**



**Example 1:**

```
Input: x = 123
Output: 321
```

**Example 2:**

```
Input: x = -123
Output: -321
```

**Example 3:**

```
Input: x = 120
Output: 21
```



**Constraints:**

- $-2^{31}$ <= x <= $2^{31} - 1$

## 2) Solution

### ãƒ»Math

- Time Complexity: $$ O(log(x)) $$
- Space Complexity: $$ O(1) $$

#### a. C++

- Runtime: 0 ms
- Memory: 8.30 MB

```C++
class Solution {
public:
    int reverse(int x) {
        long ans = 0;

        while(x != 0){
            ans = ans * 10 + x % 10;
            x /= 10;
        };

        // the value to go outside the signed 32-bit integer range [-2^31, 2^31 - 1],
        // then return 0.
        return (ans < INT_MIN || ans > INT_MAX)? 0 : ans;
    }
};

```

#### b. Go

- Runtime: 0 ms
- Memory: 3.97 MB

```go
func reverse(x int) int {
    ans := 0

    for x != 0 {
        digit := x % 10
        x /= 10

        // MaxInt32 is 2^31 - 1 = 2147483647
        if ans > math.MaxInt32/10 || (ans == math.MaxInt32/10 && digit > 7) {
            return 0
        } 

        // MinInt32 is -2^31 = -2147483648
        if ans < math.MinInt32/10 || (ans == math.MinInt32/10 && digit < -8) {
            return 0
        }

        ans = ans * 10 + digit
    }

    return ans

}
```

#### c. Python

- Runtime: 45 ms
- Memory: 19.20 MB

```python
class Solution:
  def reverse(self, x: int) -> int:
    ans = 0
    sign = -1 if x < 0 else 1
    x *= sign

    while x:
      ans = ans * 10 + x % 10
      x //= 10

    return 0 if ans < -2**31 or ans > 2**31 - 1 else sign * ans
```
