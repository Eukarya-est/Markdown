# 9. Palindrome Number

## 1. Question

Given an integer `x`, return `true` *if* `x` *is a* ***palindrome****, and* `false` *otherwise*.

**Example 1:**

> **Input:** x = 121
> 
> **Output:** true
> 
> **Explanation:** 121 reads as 121 from left to right and from right to left.

**Example 2:**

> **Input:** x = -121
> 
> **Output:** false
> 
> **Explanation:** From left to right, it reads -121. From right to left, it becomes 121-. Therefore it is not a palindrome.

**Example 3:**

> **Input:** x = 10
> 
> **Output:** false
> 
> **Explanation:** Reads 01 from right to left. Therefore it is not a palindrome.

**Constraints:**

- $-2^{31}$ <= x <= $2^{31} - 1$

## 2. Solution

### 1) Math

- Time Complexity: $$ O(log(x)) $$
- Space Complexity: $$ O(1) $$

#### A. C++

- Runtime: 0 ms
- Memory: 8.59MB

```C++
class Solution {
public:
    bool isPalindrome(int x) {
        if (x < 0) return false;

        int y = x;
        long reversed = 0;

        while (y > 0) {
            reversed = reversed * 10 + y % 10;
            y /= 10;
        }

        return reversed == x;
    }
};
```

func isPalindrome(x int) bool {

    }

#### C. Python

- Runtime: 8 ms

```python
class Solution:
    def isPalindrome(self, x: int) -> bool:
        if x < 0:
            return False

        y = x
        reverse = 0

        while y > 0:
            y, digit = divmod(y, 10)
            reverse = reverse * 10 + digit
        
        return x == reverse
```
### 2) String

- Time Complexity: $$ O(log(x)) $$
- Space Complexity: $$ O(1) $$

#### A. C++

- Runtime: 4 ms
- Memory: 10.86MB

```C++
class Solution {
public:
    bool isPalindrome(int x) {
        if (x < 0) return false;

        const string s = to_string(x);
        int l = 0;
        int r = s.length() - 1;

        while (l < r) {
            if (s[l] != s[r]) {
                return false;
            }
            l++;
            r--;
        }

        return true;
    }
};
```
