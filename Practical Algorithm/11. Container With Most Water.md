# 11. Container With Most Water

#Medium #Array #Greedy #Two pointer

[toc]

## 1. Question

You are given an integer array `height` of length `n`. There are `n` vertical lines drawn such that the two endpoints of the `ith` line are `(i, 0)` and `(i, height[i])`.

Find two lines that together with the x-axis form a container, such that the container contains the most water.

Return *the maximum amount of water a container can store*.

**Notice** that you may not slant the container.

**Example 1:**

![q11](.\imgs\question_11.jpg)

> **Input**: height = [1,8,6,2,5,4,8,3,7]
> **Output**: 49
> **Explanation**: The above vertical lines are represented by array [1,8,6,2,5,4,8,3,7]. In this case, the max area of water (blue section) the container can contain is 49.

**Example 2:**

> **Input**: height = [1,1]
> **Output**: 1

**Constraints:**

- $$ n == height.length $$
- $$ 2 <= n <= 10^5 $$
- $$ 0 <= height[i] <= 10^4 $$

## 2. Solution

### 1) Greedy

- Time Complexity: $$ O(n) $$
- Space Complexity: $$ O(1) $$

#### A. C++

- Runtime: 0 ms
- Memory: 63.00MB

```c++
class Solution {
public:
    int maxArea(vector<int>& height) {
        const int n = height.size();
        int maxV = 0;
        int l = 0, r = n - 1;
        
        while (l < r) {
            int hL = height[l], hR = height[r];
            
            // Calculate area using the shorter of the two walls
            int curr = min(hL, hR);
            maxV = max(maxV, (r - l) * curr);
            
            // Optimization: Move pointers and skip lines that are shorter 
            // than the current container height
            if (hL < hR) {
                while (l < r && height[l] <= hL) l++;
            } else {
                while (l < r && height[r] <= hR) r--;
            }
        }
        return maxV;
    }
};
```
