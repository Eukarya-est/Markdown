## I. Question

Given an array of integers `nums` and an integer `target`, return *indices of the two numbers such that they add up to `target`*.

You may assume that each input would have **exactly one solution**, and you may not use the *same* element twice.

You can return the answer in any order.

**Example 1:**

> Input: nums = [2,7,11,15], target = 9
> Output: [0,1]
> Explanation: Because nums[0] + nums[1] == 9, we return [0,1].

**Example 2:**

> Input: nums = [3,2,4], target = 6
> Output: [1,2]

**Example 3:**

> Input: nums = [3,3], target = 6
> Output: [0,1]

**Constraints:**

- $$ 2 \leqq nums.length \leqq 104 $$

- $$ -109 \leqq nums[i] \leqq 109 $$

- $$ -109 \leqq target \leqq 109 $$

- **Only one valid answer exists.**

## Ⅱ. Solution

### A. One-Pass Hash Table

- Time Complexity:  $$ O(n) $$
- Space Complexity: $$ O(n) $$

#### 1. C++

- Runtime: 0 ms
- Memory: 14.76 MB

```c++
class Solution {
 public:
  vector<int> twoSum(vector<int>& nums, int target) {
    unordered_map<int, int> numToIndex;

    for (int i = 0; i < nums.size(); ++i) {
      if (const auto it = numToIndex.find(target - nums[i]); // 'find': iterator
          // If Iterator 'find' doesn't reach the past of the element (better than end())
          it != numToIndex.cend()) 
        return {it->second, i}; // it->first: key, it->second: value
      numToIndex[nums[i]] = i;
    }

    throw;
  }
};
```
#### 2. Rust

- Runtime: 0 ms
- Memory: 2.64 MB

```rust
impl Solution {
    pub fn two_sum(nums: Vec<i32>, target: i32) -> Vec<i32> {
        let mut num_to_index = std::collections::HashMap::new();

        // .iter().enumerate() gives both the index (i) and a reference to the value (&num)
        for (i, &num) in nums.iter().enumerate() {
            let complement = target - num;

            if let Some(&prev_index) = num_to_index.get(&complement) {
                //cast `prev_index` and `i` from `usize` to `i32`
                return vec![prev_index as i32, i as i32];
            }

            num_to_index.insert(num, i);
        }
        unreachable!()
    }
}
```

#### 3. Go

- Runtime: 0 ms
- Memory: 5.90 MB

```go
func twoSum(nums []int, target int) []int {
    // preallocate map capacity
    numToIdx := make(map[int]int, len(nums)) 
    for i, num := range nums{
        // ok: map lookups, 
        // 'true' if the key exists in the map,
        if index, ok := numToIdx[target - num]; ok{ 
            return []int{index, i}
        	}
        numToIdx[num] = i
    	}
    return nil
	}
}
```

- Runtime: 0 ms

```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        num_to_idx = {}

        for i, num in enumerate(nums):
            if target - num in num_to_idx:
                return num_to_idx[target - num], i
            num_to_idx[num] = i

```
