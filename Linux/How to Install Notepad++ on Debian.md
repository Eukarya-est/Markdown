# How to Install Notepad++ on Debian

[toc]

## 1. Prerequisites: Install and Enable the Snap

1. Install the Snapd

   ```shell
   sudo apt install snapd
   ```

2. Enable the Snapd

   ```shell
   sudo systemctl enable snapd
   ```

   ```shell
   sudo systemctl start snapd
   ```

## 2. Install Notepad++

- Install the Notepad++

  ```shell
  sudo snap install notepad-plus-plus
  ```

## 3. Execute Notepad++

- Execute Notepad++

  ```shell
  notepad-plus-plus
  ```
