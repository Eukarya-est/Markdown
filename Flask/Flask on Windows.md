# Flask on Windows

[toc]

## 1. Set python environment

```shell
python -m venv [dirName]
.\env\Scripts\activate
```

## 2. Install Flask

```shell
pip3 install Flask
```

## 3. Flask project start

```shell
flask --app [name.py] run -p [PortNumber]
```

## 4. Flask-MySQL

```shell
pip3 install flask-mysql
```
